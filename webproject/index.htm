Welcome

Welcome to Video Game Console Wars. The two biggest consoles out there are the Microsoft Xbox One and the Sony Playstation 4. Each console has it's own unique features that it brings to the table. Neither consoles are bad options. Most people would be happy with either console. While the consoless are similar, each system does things a bit differntly. This web page will give you a place to find all of the inforation you need to make a decision on which console to buy. Depending on what your needs and wants are for a video game console, one might be better than the other. PS4 is a slightly more powerful machine while Xbox has more multi media features. Video Game Console Wars will educate you on all the diffences so you go home with the right console.
